copyright.

== Description ==

Content Copy Protect plugin protects your content from being copied. 

Installation and Usage

https://vimeo.com/142833636

Important Link (Support)

	http://wordpresshowtos.blogspot.com/2015/10/content-copy-protect.html

Major features of Content Copy Protect include:

*   Floating copyright message over your images with shortcode.
*   Pressing ok button on the message box reveals the image.

How to Use and Install

* Check this link for two steps involved after plugin activation and landing on page edit screen https://wordpress.org/plugins/content-copy-protect/screenshots/
* Go to the edit screen of a page where your protected images are.
* Go to visual tab and then scroll to your image on which you want to show copyright protect message
* Left click on your copyright image 
* Click on the orange color lock icon and you are done. Shortcode will be wrapped around that image.
* Do the same for all your copyright images on the page.
* once done. Save and you are done !
* See the screen shot with orange color button highlighted. You'll click on it after selecting your copyright image in the body of the edit screen.
* Now view the page to see the copyright message inplace of your image reminding the user about the copyright content. 
* Click on "Ok" button to reveal your image. 

Screenshots

See the screenshots with orange icon which you'll select after selecting your image to wrap shortcode around it and then click update page and view to see a hover message inplace of your image which will fadeout on clicking the ok button

== Installation ==

Below are the steps to install New Page Link plugin

* Upload plugin to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Go to the edit screen of a page where your protected images are.
* Go to visual tab and then scroll to your image on which you want to show copyright protect message
* Left click on your copyright image once to just select it.
* Click on the orange color lock icon on menu bar above edit screen and you are done. Shortcode will be wrapped around that image.
* Do the same for all your copyright images on the page.
* once done. Save and you are done !
* See the screen shot with orange color button highlighted. You'll click on it after selecting your copyright image in thebody of the edit screen.
* Now view the page to see the copyright message inplace of your image reminding the user about the copyright content. 
* Click on "Ok" button to reveal your image.
